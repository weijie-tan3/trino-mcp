A simple Model Context Protocol (MCP) server for Trino query engine with OAuth and Azure Service Principal (SPN) support.

## Quick Start

```bash
# Run directly (no installation needed)
uvx trino-mcp

# Or install via pip
pip install trino-mcp
```
